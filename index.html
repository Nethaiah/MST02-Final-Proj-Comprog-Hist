What is Programming Language ?

A programming language is a vocabulary and grammatical system used to give instructions to a computer or other computing device so that it can carry out particular tasks. • The term programming language usually refers to high-level languages, such as BASIC, C, C++, COBOL, Java, FORTRAN, Ada, and Pascal.

The description of a programming language is usually split into the two components of syntax (form) and semantics (meaning), which are usually defined by a formal language. Some languages are defined by a specification document (for example, the C programming language is specified by an ISO Standard) while other languages (such as Perl) have a dominant implementation that is treated as a reference. Some languages have both, with the basic language defined by a standard and extensions taken from the dominant implementation being common.

1843 - 2014

1843: Ada Lovelace’s machine algorithm

Ada Lovelace invents the first-ever machine algorithm for Charles Babbage’s Difference Machine that lays the foundation for all programming languages.

1944-45: Plankalkül

Somewhere between 1944-45, Konrad Zuse developed the first ‘real’ programming language called Plankalkül (Plan Calculus). Zeus’s language (among other things) allowed for the creations of procedures, which stored chunks of code that could be invoked over and over to perform routine operations.

1949: Assembly Language

Assembly language was used in the Electronic Delay Storage Automatic Calculator (EDSAC). Assembly language was a type of low-level programming language that simplified the language of machine code. In other words, the precise guidelines required to use a computer.

1949: Shortcode

John McCauley first proposed Shortcode (or Short-order code) as a High-Level Language (HLL) in 1949. However, it was William Schmitt who implemented it for the BINAC computer the same year and for the UNIVAC in 1950.

1952: Autocode

The term "autocode" was used to refer to a group of programming languages. Autocode, the first-ever compiled language to be implemented, was created by Alick Glennie for the Mark 1 computer at the University of Manchester. A compiler program can convert Autocode directly into machine code. Along with the Mark 1, Autocode was also used on the early Ferranti Pegasus and Sirius computers.

1957: FORTRAN

John Backus is credited with developing FORmula TRANslation, also known as FORTRAN, which is the oldest programming language still in use. The programming language was developed for complex calculations in science, math, and statistics. Some of the most cutting-edge supercomputers in the world are still running FORTRAN today.

1958: ALGOL (Algorithmic Language)

A joint committee of American and European computer scientists developed the algorithmic language, or ALGOL. Some of the most significant programming languages, including Pascal, C, C++, and Java, were first developed using ALGOL.

1958: LISP (List Processor)

John McCarthy created the list processor, also known as LISP, at the Massachusetts Institute of Technology (MIT). Originally purposed for artificial intelligence, LISP is one of the oldest programming languages still in use today and can be used in the place of Ruby or Python. The tech stacks of organizations like Acceleration, Boeing, and Genworks still use LISP.

1959: COBOL (Common Business Oriented Language)

The programming language used to power many credit card processors, ATMs, phone and cell calls, hospital signals, and traffic signal systems, to name a few, is called Common Business Oriented Language (COBOL). The development of the language was led by Dr. Grace Murray Hopper and was designed so that it could run on all brands and types of computers.

1964: BASIC (Beginner’s All-Purpose Symbolic Instruction Code)

The Beginners All-Purpose Symbolic Instruction Code, also known as BASIC, was created by a group of Dartmouth College students. The language was created for students who were not proficient in computers or mathematics. The first commercially successful offering from Microsoft was the language that Bill Gates and Paul Allen furthered developed.

1970: PASCAL

Named after the French mathematician Blaise Pascal, Niklaus Wirth developed the programming language in his honor. It was developed as a learning tool for computer programming which meant it was easy to learn. It was favored by Apple in the company’s early days, because of its ease of use and power.

1972: Smalltalk

Computer programmers could alter code on the fly thanks to Smalltalk, which Alan Kay, Adele Goldberg, and Dan Ingalls created at the Xerox Palo Alto Research Center. It introduced a number of programming language features that are present in contemporary languages like Python, Java, and Ruby.

1972: C

For use with the Unix operating system, Dennis Ritchie created the program at the Bell Telephone Laboratories. Because it was based on an earlier language called "B," it was given the name "C." Many of the most popular languages used today, including C#, Java, JavaScript, Perl, PHP, and Python, are derivations of C. It also has been/still being used by huge companies like Google, Facebook, and Apple.

1972: SQL (SEQUEL at the time)

SQL was first developed by IBM researchers Raymond Boyce and Donald Chamberlain. SEQUEL (as it was referred to at the time), is used for viewing and changing information that is stored in databases. Nowadays the language is an acronym – SQL, which stands for Structured Query Language. There are a plethora of companies that use SQL and some of them include Microsoft and Accenture.

1980/81: Ada

Ada was originally designed by a team led by Jean Ichbiah of CUU Honeywell Bull under contract to the United States Department of Defense. Ada is a structured, statically typed, imperative, wide-spectrum, and object-oriented high-level programming language that bears the name of the mathematician Ada Lovelace, who lived in the middle of the 19th century. Ada was extended from other popular programming languages at the time such as Pascal. Ada is used for air-traffic management systems in countries such as Australia, Belgium, and Germany as well as a host of other transport and space projects.

1983: C++

At Bell Labs, Bjarne Stroustrup improved the C language; C++ is an extension of C with new features like classes, virtual functions, and templates. Since 1986, it has consistently ranked among the top 10 programming languages, and in 2003, it was inducted into the Hall of Fame. C++ is used in MS Office, Adobe Photoshop, game engines, and other high-performance software.

1983: Objective-C

The primary programming language used to create software for Apple's operating systems, macOS and iOS, was created by Brad Cox and Tom Love.

1987: Perl

Perl is a general-purpose, high-level programming language that was developed by Larry Wall. Originally intended as a text editing scripting language, it is now widely used for a variety of tasks including CGI, database applications, system administration, network programming, and graphic programming.

1990: Haskell

A general-purpose programming language called Haskell is named for the American mathematician and logician Haskell Brooks Curry. It is a purely functional programming language, which means that math is at its core. It is employed in a variety of fields, particularly those involving intricate calculations, records, and number crunching. Haskell is not frequently used for well-known applications, like many other programming languages from this era.

1991: Python

Python was created by Guido Van Rossum and named after the British comedy group "Monty Python." It is a high-level, general-purpose programming language that was developed to support a range of coding approaches and be entertaining to use (a number of the tutorials, examples, and instructions frequently make Monty Python references). Companies like Google, Yahoo, and Spotify use Python, one of the most widely used programming languages in the world today.

1991: Visual Basic

Developed by Microsoft, Visual Basic allows programmers to utilize a drag-and-drop style of choosing and changing pre-selected chunks of code through a graphical user interface (GUI). The language is not overly used these days however Microsoft has used portions Visual Basic to a number of their applications like Word, Excel, and Access.

1993: Ruby

Ruby is an interpreted high-level programming language that was created by Yukihiro Matsumoto. a teaching tongue that was influenced by, among other things, Perl, Ada, Lisp, and Smalltalk. Ruby on Rails and web application development are two of its main applications. Some well-known companies that use Ruby include Twitter, Hulu, and Groupon.

1995: Java

James Gosling developed Java as a general-purpose, high-level language for an interactive TV project. It is consistently ranked among the top programming languages in the world and has cross-platform functionality. Java is used in all kinds of devices, including computers, smartphones, and parking meters.

1995: PHP

Rasmus Lerdorf created PHP, formerly known as "Personal Home Page," which is now known as "Hypertext Preprocessor." Building and maintaining dynamic web pages as well as server-side development are two of its main applications. PHP is used by some of the biggest businesses in the world, including Facebook, Wikipedia, Digg, WordPress, and Joomla.

1995: JavaScript

Brendan Eich developed the JavaScript programming language, which is primarily used for desktop widgets, PDF documents, and dynamic web development. Nearly all significant websites use JavaScript. Some well-known examples include Mozilla Firefox, Adobe Photoshop, and Gmail.

2000: C#

C#, which is based on C++ and is very similar to Java, was created at Microsoft in the hopes of fusing the computing power of C++ with the ease of Visual Basic. Almost all Microsoft products use the language, which is primarily used for creating desktop applications.

2003: Scala

Scala is a programming language created by Martin Odersky that combines organized object-oriented programming with mathematical functional programming. Due to its Java compatibility, Scala is useful for Android development. In addition to many other businesses, Scala is used by Twitter, Foursquare, Netflix, Linkedin, and many other tech giants.

2003: Groovy

Groovy was created by James Strachan and Bob McWhirter and is based on Java. The language's conciseness and ease of learning increase productivity. Starbucks, Transferwise, and Craftbase are a few well-known businesses that use Groovy in their tech stacks.

2009: Go

Google created Go to solve problems brought on by complicated software systems. Go has become popular among some of the biggest tech companies in the world, including Google, Uber, Twitch, and Dropbox, thanks to its straightforward and cutting-edge structure.

2014: Swift

Developed by Apple as a replacement for C, C++, and Objective-C, Swift was developed with the intention to be easier than the aforementioned languages and allow less room for error. Swift’s versatility means it can be used for desktop, mobile, and cloud applications. Leading language app ‘Duolingo’ launched a new app that was written in Swift.

Today:

Today: The concepts created in earlier languages served as the foundation for the computer programming languages currently in use. Many ancient languages are still in use today or serve as the basis for modern ones. The goal of the newer computer programming languages is frequently to make programmers' jobs easier. Computer programming languages will undoubtedly continue to be an essential component of modern life for a very long time to come thanks to the ongoing development of technology.

What is Web Design ?

The design of websites that are accessible online is known as web design. It covers a wide range of topics, such as search engine optimization, user interface design (UI design), and user experience design (UX design), among others. The creation and upkeep of a website require a variety of abilities and disciplines.

The term "web design" describes the layout of websites that are seen online. Instead of software development, it typically refers to the user experience aspects of website development. The primary focus of web design used to be creating websites for desktop browsers, but since the middle of the 2010s, designing for mobile and tablet browsers has gained significance.

Late 80's - 2019 and Beyond

The late 80s — the dark ages of design

Because designers were forced to use black screens with pixels during this period of web design, it was called the "dark ages." In this mock-up of what Google might have looked like in the 1980s, you can see how designs were composed of symbols and tabulation (also known as the TAB key).

1991 — The first web page

Given his busy schedule, it's easy to understand why Berners-Lee is regarded as the father of web design. With the aid of his team, he established the first web page on August 6, 1991. This was a straightforward page with information about the World Wide Web that was hosted on a NeXT computer at the European Organization for Nuclear Research, or CERN.

1992 — The first photo on the internet

Before famous people "broke the internet" or viral memes, it all began with a seemingly unremarkable image. Les Horribles Cernettes, a pop parody band, were the stars of the first ever internet photo in 1992. This photograph of the band backstage was taken by an IT developer when Berners-Lee's team tested an update to his browser that could support photos.

1994 — The first web page

The internet started to develop into the commercial juggernaut it is today two years later. AT&T, a technology company, bought the first banner advertisement on HotWired.com in 1994. 44% of those who saw this advertisement clicked on it, and it unquestionably generated buzz among both advertisers and consumers.

1995 — Javascript

Javascript, arguably the most significant web programming language ever created, was created in the middle of the 1990s. LiveScript, formerly known as Mocha, was created to enable Netscape website developers to automate behaviors in their web design. In essence, it converts a static page into a usable one so that users don't have to wait for a new page to load for each request. Interactive elements on websites, including drop-down menus and embedded forms, were made obsolete by the new scripting technology.

1996 — The rise of Flash

Here comes FutureSplash, which became Adobe Flash. Web designers now have the freedom to incorporate animation, video, and audio to make more dynamic websites thanks to this ground-breaking technology.

1998 — The invention of CSS

CSS, a foundational piece of web design technology, was created in the late 1990s. This programming language, also referred to as Cascading Style Sheets, was developed in December 1996 by Hakon Wium Lie and Bert Bos. For the first time, the visual components (layouts, colors, and font) could be distinguished from the content (HTML). This gave website designers more freedom and power over how a website looked than ever before.

2003 — The birth year of blogging

WordPress first appeared in 2003. The blogging application, developed by Matt Mullenweg and Mike Little, was a development of an earlier venture called Cafelog. Users could only initially create straightforward, static blog posts with very little visual customization.

2004 — The rise of MySpace

MySpace not only popularized coding but also served as the first significant social media platform. The platform allowed users to edit their layout or "wall" using basic HTML programming.

2007 — The beginning of the mobile era

Making websites mobile-friendly first became a priority in 2007 with the introduction of the first iPhone. In order to meet the demand for browsing on mobile devices, 960-grid systems and 12-column division became more common in design.

2010 — The rise of responsive design

In response to the rising popularity of smartphone browsing, responsive design was created. Regardless of the device you're using to browse, this technology enables web designers and developers to build websites that automatically adapt and scale to any size.

2010-2019 — Modern web design

The field of web design and development has grown rapidly, attracting individuals from a wide range of generations, cultures, and occupations. This has led to a wide variety of aesthetics and styles, some of which have even been influenced by earlier iterations of web design! This individualism has only increased as a result of web design's high degree of customization.

2019 and beyond — The future of web design

Today, browsing a website is a multi-sensory experience, thanks to the growing popularity of full-screen video backgrounds, animation, and interactive navigation. However, this is only the tip of the iceberg. As we move into the future of web design, it's likely that users will be more involved than ever.

History and background
of Web Development :

While employed at CERN in 1989, British scientist Tim Berners-Lee created the World Wide Web (WWW). The Web was originally conceived and developed to meet the demand for automated information-sharing between scientists in universities and institutes around the world.

Instead of being a solitary laboratory, CERN serves as the hub for a large community of more than 17 000 scientists from over 100 different nations. The scientists typically work at universities and national laboratories in their home countries, though they frequently spend time at CERN. Therefore, having trustworthy communication tools is crucial.

How the Web started :

In March 1989 and May 1990, respectively, Tim Berners-Lee produced the first and second proposal for the World Wide Web. Robert Cailliau, a Belgian systems engineer, and I formalized this as a management proposal in November 1990. The main ideas and key terms underlying the Web were outlined in this. The document described a "hypertext project" called "WorldWideWeb" in which "browsers" could view a "web" of "hypertext documents."

Tim Berners-Lee demonstrated his concepts by having the first Web server and browser operational at CERN by the year's end in 1990. On a NeXT computer, he wrote the code for his Web server. The computer was marked with a handwritten label in red ink that read: "This machine is a server. DO NOT POWER IT DOWN!!"

info.cern.ch was the URL of the first website and Web server ever created, which was located on a NeXT computer at CERN.

In addition to a description of hypertext, technical information on building a Web server, and links to other Web servers as they became available, this page contained links to information about the WWW project itself.

The WWW's layout made it simple to access already-existing data, and an early web page linked to data that CERN scientists could use (such as the CERN phone book and instructions for using the facility's main computers). There were no search engines in the early years, so a search facility relied on keywords.

On NeXT computers, Berners-Lee's original web browser demonstrated his vision and included many modern web browser features. It also featured the first Web editing feature: the ability to edit pages directly from within the browser. On a NeXT computer in 1993, the browser can be seen open in this screenshot.

Becoming global

The Stanford Linear Accelerator Center (SLAC) in California, another particle physics lab, hosted the first Web server in the US in December 1991, thanks to the efforts of Paul Kunz and Louise Addis. There were essentially only two types of browsers available at this time. One was the sophisticated original development version, which was only compatible with NeXT computers. The other was the 'line-mode' browser, which was simple to set up and use on any platform but had a low level of functionality and usability. The small CERN team was unable to complete the necessary work to advance the system, so Berners-Lee made an online appeal for additional programmers to join the effort.

The University of Illinois' National Center for Supercomputing Applications (NCSA) debuted the first iteration of its Mosaic browser at the beginning of 1993. This program offered amiable window-based interaction and operated in the X Window System environment, which is well-liked in the research community. The NCSA soon after released versions for both the PC and Macintosh operating systems. The proliferation of the WWW was immediately impacted by the availability of dependable, user-friendly browsers on these widely used computers. On 30 April 1993, CERN made the source code of WorldWideWeb available on a royalty-free basis, making it free software. By late 1993 there were over 500 known web servers, and the WWW accounted for 1% of internet traffic, which seemed a lot in those days (the rest was remote access, e-mail and file transfer). 1994 was the “Year of the Web”. Initiated by Robert Cailliau, the First International World Wide Web conference was held at CERN in May. It was attended by 380 users and developers, and was hailed as the “Woodstock of the Web”.

As 1994 went on, news about the Web began to circulate. The International WWW Conference Committee (IW3C2) and the NCSA jointly organized a second conference in the US in October, which drew 1300 attendees. The Web had 10,000 servers, 2000 of which were commercial, and 10,000,000 users by the end of 1994. Traffic was equivalent to shipping the entire collected works of Shakespeare every second. Technology was continuously improved to meet evolving needs. Security and tools for e-commerce were the most important features soon to be added.

What is Wed Development?expand_more

Web development, also known as website development, refers to the tasks associated with creating, building, and maintaining websites and web applications that run online on a browser. It may, however, also include web design, web programming, and database management.

What is Web Developer?expand_more

A web developer is a programmer who makes websites, and apps for the World Wide Web. A web developer builds a website from the bottom up, developing everything from layout to functions and features on the page.

What is Web Design?expand_more

Are a web developer and a web designer the same thing?expand_more

A web developer is more concerned with functionality and features of a website, and versed in the programming languages required to create them. Web developers will still need to understand the aesthetics and art direction of the site as they implement features.

A web designer will create the logos, graphics, and layout that determine the look and feel of a website. Web designers will use software like Adobe Photoshop and Illustrator to create these materials. A web designer with a knowledge of code can help him/her better communicate with the web developer.

What are types of Web developer?expand_more

There are three kinds of web developer specialization:

Front-end developers are responsible for behaviour and visuals that run in the user browser.

Back-end developers deal with the servers.

Full-stack developers are responsible for both.
